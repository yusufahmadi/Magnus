2 Master

Master adalah data yang digunakan untuk mendukung sebuah laporan, input data transaksi dan proses data.

Berikut adalah Fungsi/Perintah2 yang umum pada Modul Master

Fungsi pada form master data:

1.       Baru                       : Menginput master baru.

2.       Ubah                     : Merubah/edit data yang sudah ada.

3.       Hapus                   : Saat pertamakali dilakukan maka fungsi ini akan membuat data menjadi tidak aktif (dapat di aktifkan kembali),

                  Jika di lakukan Hapus pada data yang tidak aktif maka akan di hapus secara permanen dari database.

4.       Refresh                : Merefresh / Memanggil data master terbaru di database.

5.       Export                   : Export Ke File xls (Excel).

6.       Cetak                    : Mencetak atau Export ke Pdf.

Menu pada form entri data :

1.       Save                      : Menyimpan data dan form entrain tetap terbuka ().

2.       Save And Exit     : Menyimpan data kemudian form entrain akan tertup untuk kembali ke form master data.

3.       Save And New   : Menyimpan data kemudian form entrain akan di kosongkan untuk input data baru.

4.       Reset Change    : Mengembalikan date ke sebelum di rubah. (tidak berlaku ketika sudah melakukan Save).

7.       Delete                  : Saat pertamakali dilakukan maka fungsi ini akan membuat data menjadi tidak aktif (dapat di aktifkan kembali),

                  Jika di lakukan Hapus pada data yang tidak aktig maka akan di hapus secara permanen dari database.

5.       Close                     : Menutup form entrain kembali ke form master data.    

 

2.1 Master Barang

Kode Kategori   : Menunjukan Kategori Barang, (Untuk Melihat daftar Kategori, Menambah, Mengubah, Menghapus Kategori Buka Menu Master Kategori)

Kode                     : Kode Barang (Unik) (Max 13 Karakter)

Nama                    : Nama Barang (Max 50 Karakter)

Catatan                : Catatan Item (Max 150 Karakter)

Satuan                  : Satuan Barang

IsActive                : Menunjukan Status Barang Aktif

Harga Beli           : Harga Beli Barang (Hanya Supervisor yang dapat melihat ini)

Tanggal Buat     : Tanggal pertama kali entri barang

User Buat            : User yang mengentri pertama kali

Tanggal Ubah   : Tanggal perubahan terakhir

User Ubah          : User yang merubah terakhir kali
